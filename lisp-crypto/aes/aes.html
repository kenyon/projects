Common Lisp AES encryption

I have made two implementations of AES
  • aes32.lisp is optimized for CLs that handle 32 bits arithmetic, such as CMUCL.
    A program to make NIST test vectors is aes-nits.lisp.
  • aes8.lisp is based on the reference implementation and uses 8 bits arithmetic.
    The performance is slow but less varying among Common Lisp implementations.

Download: aes.tgz.
